# USAGE
Tool created for a Node.js training.

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -n, --numero   Number to be multiply by                    [number] [required]
  -p, --print    Print the result of the multiply     [boolean] [default: false]
  -l, --limit    Set the limit to be multiply by          [number] [default: 10]
  ```
